Scafold a fullstack project for use [Tealina](https://www.tealina.dev)

### Usage

```bash
pnpm create tealina
```

### What it does?
1. Create a backend project with conventional folder structure,
the structure(one API, one file) make APIs type extrable.
with tealina CLI, you can generate APIs document, and align the API files in convention.

2. Create a frontend project base on `create-vite`'s typescript template, auto inject an api folder, for easy to use the backend APIs with type safety,
